# Modelowanie Klasy Niewydolności Serca (NYHA)

##  Cel projektu

Celem tego projektu było zbudowanie i porównanie modeli uczenia maszynowego do przewidywania stopnia niewydolności serca (w 4-stopniowej skali NYHA) na podstawie danych klinicznych, laboratoryjnych i wyników testów wysiłkowych pacjentów.

Projekt demonstruje pełen proces analizy danych, od czyszczenia i przygotowania surowego zbioru, przez analizę eksploracyjną (EDA), po budowę, ocenę i interpretację modeli klasyfikacyjnych.
